**Truck**主要字段

| 字段 | 说明 |
| Plate Number | 客户主体官方名称，唯一性 |
| Truck Type | 客户标签，简称，唯一性 |
| Van Type | 车厢类型，包含Dry，Reefer共2种 |
| Registration Number | 车辆注册证书号码 |
| Gross Capacity | 毛重容量 |
| Net Capacity | 净重容量（等于常规“载重”） |
| Volume | 容积 |
| Moedl | 车辆型号 |
| Status | 车辆状态 |
| Vendor | Truck所属供应商 |
| Creation Time | 创建时间 |
