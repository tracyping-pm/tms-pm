- Waybill No.：识别运单
- 若同一个运单出现在Sheet1,2,3中，1中进行正常处理，则2认为该运单已重复，不再处理该运单

- 若填写则C-F列均需填写完全

- G-J列，Track 信息：默认按该运单路线起始地与目的地记录对应四个记录，多次上传进行增量更新
- G列为：Arrival at Origin，地址取路线起始地，时间取G列时间
- H列：Loading Completion，地址取路线起始地，时间取H列时间

- J列：Unloading Completion，地址取路线目的地，时间取J列时间
- 允许4列皆为空，或任意N项有值，格式为MM/DD HH:MM
- 格式不正确，进行报错，Column X format error
- K列：Customer Code, 格式为type:code,   多次上传进行增量更新  Add
- 格式不正确，进行报错，Column K format error
- 若type 为空或错误Type, 报错  Column K Type error
- L列：POD link  【备注：培训需告知业务POD文件夹权限需分享】
