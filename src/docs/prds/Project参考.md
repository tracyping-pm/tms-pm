**Project**主要字段

| 字段 | 说明 |
| Project Name | 项目名称，具有唯一性 |
| Customers | 客户名称 |
| Tag | 客户简称 |
| Commodity | 商品类型 |
| Project Status | 项目状态 |
| Transportation Status | 项目运输状态 |
| Financial Status | 项目财务状态 |
| Days for POD | 回单要求回传天数 |
| Creation Time | 项目创建时间 |
| Agreed Start Time | 项目合同约定开始时间 |
| Agreed End Time | 项目合同约定结束时间 |
| Completed Time | 项目结束时间 |
